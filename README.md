# 📝 TODOs — Learning Vite + React + TypeScript

This project is a **learning playground** for building web apps with **Vite**, **React**, and **TypeScript** from scratch. It started as the default Vite React-TS template and evolved as I learn new concepts.

## 🎯 What I'm Learning

- **Vite** — fast dev server with Hot Module Replacement (HMR)
- **React** — components, state (`useState`), props, JSX
- **TypeScript** — interfaces, type annotations, generics
- **CSS** — component styling, CSS variables, dark mode

## 🧱 Current App: Todo List

A simple todo app with:
- Add / toggle complete / delete todos
- Auto-sort: incomplete at top, completed at bottom (by completion time)
- Controlled input
- Clean dark/light mode support

## 🚀 Commands

| Command | Description |
|---|---|
| `npm run dev` | Start dev server with HMR |
| `npm run build` | Type-check + production build to `dist/` |
| `npm run preview` | Preview production build |
| `npm run lint` | Run oxlint |
